## ビンゴゲームのカード作成
### ▼参考サイト
- https://blog.jnito.com/entry/2019/05/03/121235
    - ビンゴカード作成問題(良いトレーニングになりました。感謝🙏)
### ▼仕様(上記参考サイトより引用)
- ビンゴカードの数字のルール
    - B：1～15のどれか
    - I：16～30のどれか
    - N：31～45のどれか
    - G：46～60のどれか
    - O：61～75のどれか    
- プログラムの要件
    - 毎回異なるカードを生成します。
    - 各列はパイプ（|）で区切ります。
    - 数字や"BINGO"の文字は右寄せで出力します。
    - 真ん中（FREEになる場所）はスペースを出力します。
